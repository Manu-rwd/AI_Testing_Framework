Regula "tipuri_de_continut":
    {
    Tipurile de continut sunt:
        1. Continut tip Tabel -> este utilizat pentru organizarea datelor in randuri si coloane. Contine obligatoriu notiunile de table (Thead, Tr, Td). In cadrul userstory-ului se regaseste descris sub denumirea de "column".
        2. Continut tip Lista -> este reprezentata afisarea elementelor intr-o ordine secventiala, fie ordonata, fie neordonata. In cazul ERP-ului este reprezentata de meniul de accesare din partea stanga.
        3. Continut tip Formular -> este utilizat pentru introducerea/modificarea informatiilor si include obligatoriu un buton de trimitere date (Adauga, Modifica, Salveaza, Importa, Exporta)
        4. Continut tip Document -> reprezentata de un tot ceea ce nu este un tabel lista sau formular. Poate contine paragrafe, titluri, imagini sau link-uri.
        }
